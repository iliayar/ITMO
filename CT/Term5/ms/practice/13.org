- \(X_n\) --- рекурентное соотношение
\[ X_n + a_{n - 1} X_{n - 1}  + \dots + a_0 X_0 = 0\]

* Алгоритм
Каждому корню характерисчтиеского урванения сопоставляем несколько
решений по жтому принципц. Общее решение является линейной комбинацией
этих решений.

#+begin_examp org
Уравнение второго порядка
\[ x_2 + bx_1 + cx_0 = 0 \]
1. \(D > 0\), \(\lambda_1 \neq \lambda_2\) --- два вещественных корня
   \[ x_n = C_1 \lambda_1^n + C_2 \lambda_2^n \]
2. \(D = 0\), \(\lambda_1 = \lambda_2 = \lambda\)
   \[ x_n = C_1 \lambda_^n + C_n n \lambda^n \]
3. \(D < 0\), \(\lambda_{1, 2} = 2 \todo\)
#+end_examp
